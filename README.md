# Prima PR Demo

Repository di prova creato per fare pratica con il flusso di una Pull Request su GitHub.

## Cosa contiene

Un piccolo script Python di esempio (`saluto.py`) con una funzione di saluto.

## Progetto videogioco — *Lanterna*

Il repo ospita anche lo sviluppo di un piccolo gioco 3D cozy con una rete di
agenti Claude Code come team di sviluppo:

- **`.claude/agents/`** — il team: `game-designer`, `gameplay-programmer`,
  `level-designer`, `concept-artist` (il 3D artist è umano 🙋)
- **`docs/GDD.md`** — il Game Design Document
- **`docs/PIANO-DI-PRODUZIONE.md`** — Gantt a 12 settimane, milestone e workflow
